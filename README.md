# Maximaleffekt Tap

## How do I install these formulae?

`brew install maximaleffekt/tap/<formula>`

Or `brew tap maximaleffekt/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "maximaleffekt/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
