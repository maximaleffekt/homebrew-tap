# Maximaleffekt Tap

My personal tap for packaging software that isn't available in homebrew directly.

## Available formulae and casks

- [NegPy](https://github.com/marcinz606/NegPy) - Python based film negative scanner and editor

## How do I install these formulae?

`brew install maximaleffekt/tap/<formula>`

Or `brew tap maximaleffekt/tap` and then `brew install <formula>`.
